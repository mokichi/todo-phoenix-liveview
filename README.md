## Prerequisites

We recommend to install following with [asdf](https://asdf-vm.com/#/). Also see [.tool-versions](.tool-versions).

- Erlang
- Elixir
- Node.js

### Databases

- PostgreSQL 13

If you are using Docker, see [docker-compose.yml](docker-compose.yml).

## Setup

```
$ mix setup
```

## Start Phoenix server

```
$ mix phx.server
```

http://localhost:4000
